オートファジー

日常でできるセルフケア習慣まとめ｜口内炎・高血圧・頭痛・目の疲れまで

「体の小さな不調」を放置しない習慣が大切 日常生活で感じる「なんとなく体が重い」「よく口内炎ができる」「雨の日に頭が痛い」——こういった小さな不調は、見過ごされがちです。 しかし、こういった体のサインの多くは、生活習慣の見直しや簡単なセルフケアで改善できることが多いです。この記事では、体のさまざまな小さな不調への対処法を一まとめで紹介します。 口内炎の原因と対処法 口内炎の最も一般的な原因の一つがビタミンB群の不足です。 ビタミンB群（特にB2・B6）が不足すると、口腔内の粘膜が弱くなり、些細な刺激でも傷がつきやすくなります。この傷から細菌が侵入して口内炎へと発展するパターンが多く見られます。 ビタミンB2を多く含む食品 食品 特徴 牛・豚レバー 含有量が非常に高い 魚類（さば・さんま） 毎日摂りやすい 納豆 手軽で継続しやすい 乳製品・卵 タンパク質も同時に補給 ビタミンB6を多く含む食品 食品 特徴 牛レバー 最も高含有 鶏のささみ 低脂肪・高タンパク まぐろ 刺身・缶詰で手軽に ピスタチオ・ひまわりの種 スナック感覚で摂れる にんにく 日常の料理に加えやすい 口内炎が頻繁にできる方は、まずビタミンB2・B6の豊富な食品を意識して摂ることから始めてみましょう。サプリメントでの補充も有効ですが、食事から摂るのが理想的です。 高血圧の危険サイン——放置してはいけない数値とは 「血圧が高くても体調が悪いとは感じにくい」——これが高血圧の最も怖い点です。 高血圧は自覚症状がほとんどないまま進行し、ある日突然「脳幹出血」「心筋梗塞」「脳卒中」を引き起こすことがあります。 危険な数値の目安 状態 収縮期（上） 拡張期（下） 正常値 120未満 80未満 要注意 130〜139 85〜89 高血圧 140以上 90以上 上が140mmHg、下が90mmHgを超えたら医師に相談することを強くおすすめします。 血圧が下がらない場合の注意 血圧の薬を飲み始めると「一生飲まなければいけない」と思い込む方もいますが、そうではありません。スポーツや体重管理（ダイエット）によって血圧が下がれば、薬を減らしたり不要になる可能性もあります。 まずは生活習慣の改善（減塩・禁煙・運動）を試み、それでも改善しない場合に薬物療法を検討するという順序が理想的です。 雨の日に頭痛がする理由と対策 「雨の日になると頭が痛くなる」という方は少なくありません。これは低気圧の変化が耳の周りの神経に影響することで起きます。 低気圧になると外部の気圧が下がり、体内（特に耳の中）との気圧差が生じます。このとき耳周辺の内耳という器官が過剰反応し、めまいに似た感覚や頭痛を引き起こします。 対処法 耳周りの血行を改善する 耳を軽くつまんで引っ張る・ゆっくり回すストレッチが有効とされています。血流改善により内耳の圧力変化への対応が良くなります。 血行を良くする食材を摂る 生姜・にんにく・玉ねぎなど血行促進効果がある食材を摂ると効果的な場合があります。 市販薬の活用 「五苓散（ごれいさん）」は低気圧頭痛に効果があるとされる漢方薬で、ドラッグストアで購入できます。 雨の予報の前日夜から水分を多めに摂ることで体内の水分バランスを整え、症状を緩和できる場合もあります。 目の体操で疲れ目を予防する 現代はパソコン・スマホを長時間使う機会が多く、目の疲れや視力低下に悩む方が増えています。 効果的な目の体操の手順 顔を動かさずに目だけを右端のギリギリまで動かし、10秒キープ 同様に左端へ動かし10秒キープ これを2セット繰り返す 同じように上下・対角線（右上と左下、左上と右下）の方向で行う 最後に両目を右回り・左回りにそれぞれ5回転させる この体操は1回5〜10分でできます。長時間のデスクワーク後や、就寝前のリラックスタイムに取り入れるのがおすすめです。 ...
